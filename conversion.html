Conversion


This program converts a number from one unit of measurement to another. The user can select the conversion type and enter a number to be converted. The result will be displayed in the result field.

Conversion Type Formula
Celsius to Fahrenheit °F = (°C × 9/5) + 32
Fahrenheit to Celsius °C = (32°F − 32) × 5/9
Meters to Feet ft ≈ m × 3.28084
Feet to Meters m ≈ ft / 3.28084

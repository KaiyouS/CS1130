Compute Grade


This program computes the grade of a student based on the score, number of items, and base grade. The program also displays the corresponding letter grade and a comment based on the computed grade.

To use the program, simply enter the score, number of items, and base grade in the input boxes below. The program will then compute the grade and display the result in the "Grade" box.

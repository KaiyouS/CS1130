Loops


This program performs different operations depending on the selected operation type. Each operation uses different types loops to get the appropriate result. Factorial uses a while loop. Sum uses a do-while loop. Average uses a for loop.

To use the program, select the operation type and enter a number. The program will then compute the result and display it in the result field.

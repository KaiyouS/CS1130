Conversion

This program converts a number from one unit of measurement to another. The user can select the conversion type and enter a number to be converted. The result will be displayed in the result field.

Conversion Type Formula
Celsius to Fahrenheit °F = (°C × 9/5) + 32
Fahrenheit to Celsius °C = (32°F − 32) × 5/9
Meters to Feet ft ≈ m × 3.28084
Feet to Meters m ≈ ft / 3.28084



Income Tax Calculator

Knowing your income tax is important. It helps you plan your finances and make better financial decisions. This program computes your income tax based on the tax table provided by the Bureau of Internal Revenue (BIR).




Loops

This program performs different operations depending on the selected operation type. Each operation uses different types loops to get the appropriate result. Factorial uses a while loop. Sum uses a do-while loop. Average uses a for loop.




Simple Payroll

This program computes the salary of an employee based on the number of days worked and the daily rate. The program also computes the gross and net pay. The program displays the data in a tabular format.

Shopping Cart

This program simulates a shopping cart. The user can select an item from the dropdown menu and enter the quantity of the item to be added to the cart. The program will then compute the total price of the item and display the result in the "Total Price" box. The program also displays the total price of all the items in the cart in the "Total" box.
